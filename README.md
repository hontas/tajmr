tajmr
=====
> Record time sometime [tajmr](https://tajmr.herokuapp.com/)

Press play to start recording time. Then pause it. Add a note if you want.
Serverless - uses firebase.io.
![Tajmr](tajmr.png)

## dev
```shell
npm run watch
```

## deploy

- Automat deploy to Heroku
- Gh-pages deployable like:
```shell
npm run deploy
```
